# Installation:
> [!NOTE]
> You'll need an ASI loader and OpenIV.asi, both can be installed in OpenIV.
1. Download the [latest release]()
2. Open OpenIV and enable Edit mode
3. Drag the .oiv file to OpenIV
4. Hit Install
5. To "mods" folder
6. Horray
